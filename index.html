Airport Distance Project

The FAA allows pilots to log flight time as Cross Country when flying to an airport that is further than 50nm away. This notebook was developed with the intention of helping the user perform that task, but allows flexibility to adjust parameters to vary the request. The airport data is pulled from the vega_datasets library and input into a pandas dataframe where the distance from the selected airport to each airport is calculated. This distance is then used to determine which airports are returned and displayed in the bar chart.

FAA Expiring Registration Numbers (N Numbers)

This project focused on using pandas dataframes to display database information downloaded daily from the FAA. The user is able to filter the data to see aircraft registration numbers that are purging from the system and becoming available to reserve.
